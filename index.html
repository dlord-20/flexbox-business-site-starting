Surrogate Formulation Software

Why MolEvo?

A massive amount of data is produced every day from analytical laboratories worldwide. A large portion of this data is aimed to characterize complex liquid mixtures.

Value Proposition

MolEvo is a software solution that allows for transforming complex and indecipherable feedstock chemical analysis into a discrete ensemble of molecules.

MolEvo Uniqueness

MolEvo uses a proprietary extension of the evolutionary algorithm allowing the user to create customized datasets for a variety of applications.
